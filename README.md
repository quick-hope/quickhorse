# QuickHorse

A fast, lightweight coding-agent CLI written in Go.

## Features

- Multiple LLM providers (OpenAI, Anthropic, Gemini, Ollama)
- MCP (Model Context Protocol) support
- Tool execution (Bash, File, Web, etc.)
- Session management
- Single binary deployment

## Install

```bash
go install github.com/quick-hope/quickhorse@latest
```

## Usage

```bash
quickhorse
```

## Providers

- OpenAI (GPT-4, GPT-4o)
- Anthropic (Claude 3.5, Claude 4)
- Gemini (Gemini Pro)
- Ollama (Local models)
