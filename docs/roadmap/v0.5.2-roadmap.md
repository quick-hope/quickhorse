### 2. Hooks System (P0)

**目标**: 生命周期钩子执行用户脚本

| HookEvent | 说明 | 触发时机 |
|-----------|------|----------|
| SessionStart | 会话开始 | 启动时 |
| SessionEnd | 会话结束 | 退出时 |
| MessageSubmit | 发送消息 | 发送前 |
| ToolCallBefore | 工具调用前 | 执行前 |
| ToolCallAfter | 工具调用后 | 完成后 |
| ModeChange | 模式切换 | 切换时 |
| OnError | 错误发生 | 错误时 |
| ShellEnv | Shell 环境 | exec_shell 前 |

**参考**: `~/ai-project/DeepSeek-TUI/crates/tui/src/hooks.rs`

**配置**:
```toml
[[hooks.hooks]]
event = "session_start"
command = "~/.quickhorse/hooks/start.sh"
timeout = 30
```

### 3. Context Compaction (P0)

**目标**: 长对话自动压缩，节省 token

| 配置 | 默认值 | 说明 |
|------|--------|------|
| enabled | true | 启用压缩 |
| token_threshold | 800000 | 触发阈值 |
| auto_floor_tokens | 500000 | 最小阈值 |
| cache_summary | true | 缓存摘要 |

**触发条件**:
- total_tokens > token_threshold
- total_tokens > auto_floor_tokens

**压缩流程**:
1. 调用 LLM 生成旧消息摘要
2. 替换旧消息为摘要
3. 保留最近 N 条消息

**参考**: `~/ai-project/DeepSeek-TUI/crates/tui/src/compaction.rs`

### 5. Command Safety 增强版 (P1)

**目标**: 增强命令安全分析

**COMMAND_ARITY 字典** (部分):
```
git add      → 2
git checkout → 2
npm run      → 2
docker compose → 2
cargo check  → 1
rm           → 1
```

**危险命令拦截**:
- `rm -rf /`
- `sudo rm`
- `mkfs`
- `dd if=/dev/zero`

**参考**: `~/ai-project/DeepSeek-TUI/crates/tui/src/command_safety.rs`
