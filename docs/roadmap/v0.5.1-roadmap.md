## 参考问题

### Resize 问题分析

**现象**: 终端宽度调整后，内容截断不换行

**根因**: `wrap_text()` 使用 word-based 换行，中文和长单词无法正确处理

**解决**: 改用 grapheme-based 换行，每个字符单独计算 width

### Warnings 分类

**Future Use**: 预留 API，暂未集成到主流程
- `ToolCardWidget` - 计划用于 transcript 工具卡片
- `FooterWidget::ready/streaming/working` - 计划用于状态切换
- `errors.rs` functions - 计划用于错误分类显示

**Technical Debt**: 代码演进遗留
- unused imports - 删除即可
- unused variables - 可能是重构遗留

**Strategy**: 使用 `#[allow(dead_code)]` 标注预留功能，清理真正的 dead code
